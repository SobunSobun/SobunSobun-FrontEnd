## 프로젝트 소개
이웃과 함께 구매하고 나누는 지역기반 서비스 소분소분

소분소분은 1인가구를 위한 지역기반 공동구매 커뮤니티 서비스입니다.
사용자들이 스스로 주최하는 공동구매를 통해 마음맞는 이웃들을 모은 뒤
함께 장보고 나누며 새로운 연결의 가치를 제공하길 기대합니다
